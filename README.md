# ExchangeRate

Introduction

The Exchange Rates API provides endpoints to manage exchange rates between different currencies. The application uses Spring Boot for the RESTful API and MongoDB to store exchange rate data.

Technologies Used

Java 17

Spring Boot 3.0

Spring Data

Restful API

MongoDB

ExchangeAPI

Lombok

Maven
